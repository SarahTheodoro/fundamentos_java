PACOTES E CLASS





















Um package Java é um mecanismo para agrupar classes de finalidades afins ou de uma mesma aplicação. Além de facilitar a organização conceitual das classes, o mecanismo de pacotes permite localizar cada classe necessária durante a execução da aplicação. Para criar pacotes em um projeto Java pelo Eclipse, basta seguir os passos: Botão direito na pasta “src” > New > Package a janela da Figura 2 será exibida. Após colocar o nome do pacote, clique em Finish. O seu pacote aparecerá em branco, sem nada dentro, isso significa que você ainda não tem nada no mesmo. Depois, Para criar classe a partir de um pacote, basta seguir os passos: Botão direito no pacote criado > New > Class.
